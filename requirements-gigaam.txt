#        pip install torch==2.6.0 torchaudio==2.6.0 --index-url https://download.pytorch.org/whl/cu124
#   3) остальное:
#        pip install -r requirements-gigaam.txt
#   4) ffmpeg (НЕ pip): https://www.gyan.dev/ffmpeg/builds/ → bin\ в PATH (ffmpeg -version)
#
